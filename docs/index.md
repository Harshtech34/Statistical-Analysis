# Week 7: Statistical Business Analysis Documentation

## Project Overview
This project applies statistical techniques on business datasets to analyze sales trends, customer behavior, and relationships between business metrics.

## Files in Repository
- statistical_analysis.ipynb  
- business_data.csv  
- statistical_report.pdf  
- requirements.txt  
- hypothesis_tests_results.txt  

## Methodology
- Descriptive statistics
- Data visualization (histograms, heatmaps)
- Hypothesis testing (t-test, ANOVA)
- Correlation analysis
- Linear regression analysis

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Open Jupyter Notebook:
   jupyter notebook statistical_analysis.ipynb

## Results Summary
- Significant relationship between price and total sales
- Churned customers show different monthly charges compared to non-churned customers
